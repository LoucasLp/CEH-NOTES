## 🎯 Objectifs du module

À l'issue de ce module, vous serez capable de :

- ✅ Définir le Social Engineering et expliquer pourquoi l'humain est vulnérable
- ✅ Identifier les phases d'une attaque SE (Research, Target, Attack, Execute)
- ✅ Distinguer techniques **humaines** (Impersonation, Tailgating) vs **informatiques** (Phishing)
- ✅ Utiliser le Social-Engineer Toolkit (SET) pour des attaques de phishing
- ✅ Reconnaître les principes psychologiques exploités (Autorité, Urgence, Réciprocité)
- ✅ Comprendre les menaces internes (Insider Threats) et l'usurpation d'identité
- ✅ Mettre en œuvre des contre-mesures (formation, procédures, sensibilisation)

## 🧠 Mémo rapide — Techniques clés

```
HUMAINES :
  Impersonation   → Se faire passer pour quelqu'un (technicien, livreur, manager)
  Tailgating      → Suivre physiquement une personne autorisée dans une zone sécurisée
  Piggybacking    → Tailgating AVEC consentement de la personne (trompée)
  Shoulder Surfing→ Observer par-dessus l'épaule (PIN, MDP)
  Dumpster Diving → Fouiller les poubelles (documents, disques, badges)
  
INFORMATIQUES :
  Phishing        → Email générique à masse → vol de credentials
  Spear Phishing  → Email CIBLÉ (personnalisé sur la victime)
  Whaling         → Spear phishing ciblant des C-levels (CEO, CFO)
  Smishing        → Phishing par SMS
  Vishing         → Phishing par appel vocal
  
PRINCIPES PSYCHOLOGIQUES (Cialdini) :
  Autorité | Urgence | Réciprocité | Sympathie | Rareté | Preuve sociale
```
