## 🎯 Objectifs du module

À l'issue de ce module, vous serez capable de :

- ✅ Comprendre le rôle du CEH et l'éthique du hacking éthique
- ✅ Identifier les différents types de hackers et leurs motivations
- ✅ Maîtriser la triade CIA et les contrôles de sécurité
- ✅ Appréhender les 5 phases d'une attaque offensive
- ✅ Relier la Cyber Kill Chain au framework MITRE ATT&CK
- ✅ Connaître le cadre légal (CFAA, GDPR, HIPAA, ISO 27001)
- ✅ Distinguer les types de tests : Pentest, VA, Red Team, Black/White/Grey Box
