## 🎯 Objectifs du module

À l'issue de ce module, vous serez capable de :

- ✅ Comprendre le fonctionnement des IDS/IPS et leurs types de détection
- ✅ Identifier les techniques d'évasion IDS (fragmentation, TTL, encoding)
- ✅ Contourner les firewalls via le tunneling et le firewalking
- ✅ Détecter et éviter les honeypots
- ✅ Utiliser les flags Nmap pour l'évasion (−f, −D, −−source-port)
