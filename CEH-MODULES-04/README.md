## 🎯 Objectif du Module

L'énumération est la **3ème phase** du hacking éthique. Elle consiste à établir des connexions actives avec la cible pour extraire des informations précises et exploitables.

```
Footprinting → Scanning → ÉNUMÉRATION → Exploitation
```

---

## ⚡ Points Clés à Retenir

- 🔴 L'énumération crée des **connexions actives** (contrairement au scanning passif)
- 🔴 Peut être **illégale** sans autorisation préalable
- 🟡 Fonctionne principalement en **environnement intranet**
- 🟢 Résultats utilisés pour les **attaques par mot de passe** et l'exploitation
