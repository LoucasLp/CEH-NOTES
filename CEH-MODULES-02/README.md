## 🎯 Objectifs du module

À l'issue de ce module, vous serez capable de :

- ✅ Comprendre les techniques de collecte d'informations passives et actives
- ✅ Utiliser Google Dorks, SHODAN et les moteurs de recherche spécialisés
- ✅ Effectuer le footprinting DNS (WHOIS, zone transfer, records)
- ✅ Exploiter les réseaux sociaux et sources OSINT pour la reconnaissance
- ✅ Maîtriser Maltego, Recon-ng, FOCA et theHarvester
- ✅ Identifier et appliquer les contre-mesures de footprinting

## ⚡ Rappel rapide — Footprinting passif vs actif

```
PASSIF  → Aucun contact direct avec la cible
          WHOIS, moteurs de recherche, DNS public, réseaux sociaux
          Légal sans autorisation préalable
          
ACTIF   → Interaction directe avec la cible
          Scan DNS, traceroute, banner grabbing
          Nécessite une autorisation écrite
```
