## 🎯 Objectifs du module

À l'issue de ce module, vous serez capable de :

- ✅ Comprendre l'architecture d'un serveur web et sa surface d'attaque
- ✅ Identifier les vulnérabilités classiques des serveurs HTTP (Apache, IIS, Nginx)
- ✅ Exploiter les mauvaises configurations et les failles de traversée de répertoire
- ✅ Utiliser Nikto, Nmap NSE et gobuster pour l'énumération web
- ✅ Déployer des contre-mesures de durcissement serveur
