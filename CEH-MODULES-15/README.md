## 🎯 Objectifs du module

À l'issue de ce module, vous serez capable de :

- ✅ Comprendre les mécanismes de l'injection SQL
- ✅ Identifier les différents types (error-based, blind, union, time-based)
- ✅ Utiliser sqlmap pour automatiser les tests d'injection
- ✅ Bypasser des filtres et WAF avec des techniques d'obfuscation
- ✅ Appliquer les contre-mesures (prepared statements, WAF, moindre privilège)
