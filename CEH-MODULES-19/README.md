## 🎯 Objectifs du module

À l'issue de ce module, vous serez capable de :

- ✅ Comprendre les modèles cloud IaaS/PaaS/SaaS et le shared responsibility model
- ✅ Identifier les menaces cloud (CSA Top Threats, misconfiguration)
- ✅ Exploiter les misconfigurations S3 et IAM
- ✅ Utiliser AWS CLI et ScoutSuite pour l'audit cloud
- ✅ Déployer des contre-mesures cloud (CloudTrail, CSPM, MFA)
