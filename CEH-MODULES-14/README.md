## 🎯 Objectifs du module

À l'issue de ce module, vous serez capable de :

- ✅ Comprendre l'architecture des applications web et leurs composants
- ✅ Connaître l'OWASP Top 10 2021 par cœur (descriptions, exemples, contre-mesures)
- ✅ Exploiter XSS, CSRF, IDOR, clickjacking et file upload vulnérabilities
- ✅ Utiliser Burp Suite et OWASP ZAP pour des tests applicatifs
- ✅ Appliquer les contre-mesures SDL/DevSecOps
